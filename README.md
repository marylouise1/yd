# <p align="center">YD
  
  
## Getting started
 
- **Installation**

```shell
$ git clone https://github.com/marylouise1/yd.git
```

- **Change directory**

```shell
$ cd yd
```

- **Virtual Environment** `Optional`

```shell
$ pip install virtualenv
$ virtualenv venv
$ source venv/bin/activate
```

- **Install dependencies**

```shell
$ pip install -r requirements.txt
```

- **Run the app**

```shell
$ python main.py
```
